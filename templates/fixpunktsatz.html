Banachscher Fixpunktsatz

Der Fixpunktsatz von Banach ist ein Mathematischer Satz der 1922 von Stefan Banach aufgestellt wurde. Mithilfe dieses Satzes kann man die Existenz von Fixpunkten nachweisen.

Voraussetzungen:

  • ein Vollständiger Metrischer Vektorraum
  • eine abgeschlossene Menge M (keine leere Menge)
  • eine Kontraktion

Begriffserklärungen:- Vollständiger Metrischer Vektorraum:
            - Vektorraum: Raum der durch Vektoren Beschrieben wird
            - Metrischer Raum: Raum in dem zwei Punkte einen nicht Negativen Abstand haben
            - Vollständiger Raum: Jede Cauchy-Folge von Elementen dieses Raumes konvergiert
                        -  Cauchy-Folge: Folge bei der der Abstand von aufeinander folgenden Gliedern immer kleiner wird.

Kontraktion: Abbildung einer Menge auf sich selbst wobei die Abstände zweier Punkte verringern

Unter diesen Voraussetzungen gilt:
Die Abbildung des Fixpunktes entspricht dem Fixpunkt Bsp.: f(x)=x
Jeder Grenzwert für ein Element der Menge M ist der Fixpunkt

Beispiel:
Wurzelfunktion: f(x)=√x

Voraussetzungen: Vollständiger Metrischer Vektorraum: kartesisches zweidimensionales Koordinatensystem
abgeschlossene Menge M: M=[1,2] (wirdfestgelegt)

Kontraktion: Im Intervall [1,2] gilt
√x<x (Abbildung ist kleiner)

Iterationsschritte:
√2≈1,4142
√1,4142≈1,1892
√1,1892≈1,0905
...
√1=1

Der Fixpunkt der Funktion √x ist also 1

Für das gegebene Beispiel sähe ein Programm folgendermaßen aus:

def f(x):
    return x**0.5 # hier soll die Funktion definiert werden.

def fixpunkt_berechnen(startwert, toleranz=10e-7, max_iterationen=1000):

    wert = startwert
    for i in range(max_iterationen):
        iteration = f(wert)
        differenz = abs(iteration - wert)

        if differenz < toleranz:
            return iteration

        wert = iteration
    return None

fixpunkt = fixpunkt_berechnen(0.1)

if fixpunkt:
    print(f"Fixpunkt: {fixpunkt}")

else:
    print("Kein Fixpunkt mit der gegebenen Toleranz und maximalen Menge an iterationen gefunden.")
