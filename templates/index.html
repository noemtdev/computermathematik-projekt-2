Implementierung numerischer Verfahren zum Lösen von Gleichungen

Einführung (Dennis)

In vielen Bereichen der Wissenschaft
und Technik kommt es vor, dass man Gleichungen lösen muss, die analytisch nicht
oder nur sehr schwer zu lösen sind. In diesen Fällen kann man auf numerische
Verfahren zurückgreifen, die die Lösung der Gleichung mit Hilfe von Computern
ermöglichen.

Numerische Verfahren lassen sich in 8
verschiedene Kategorien Gliedern, wobei in dieser Ausarbeitung nur
Verfahren aus der Kategorie der Nichtlinearen Gleichungssysteme behandelt werden.

- Bisektionsverfahren
- Banachscher Fixpunktsatz
- Newtonverfahren
- Regula Falsi Verfahren

Ein Beispiel für ein anderes Verfahren ist das Heron-Verfahren, welches für die Annäherung an die Quadratwurzel einer reellen Zahl a (a > 0) verwendet wird.

Die Iterationsvorschrift für das
Heron-Verfahren lautet:

 steht für den Näherungswert der n-ten Iteration und  steht für den Wert, von dem die Wurzel
bestimmt werden soll.

Mit Python könnte man das Heron-Verfahren folgendermaßen darstellen, allerdings ist math.sqrt ausreichend, um die Wurzel einer Zahl zu bestimmen. 

class NewtonVerfahren:
"""
Berechnung mit quadratischen Funktionen
"""
def __init__(self, a:float, b:float, c:float):
    self.a = a
    self.b = b 
    self.c = c

@classmethod
def funktionswert(self, x:float):
    return self.a*x*x + self.b*x + self.c

@classmethod
def ableitungswert(self, x:float):
    return 2*self.a*x + self.b

def verfahren(self, iterationen: int, naeherungswert: float) -> list:
    naeherungswerte = []
    for i in range(iterationen):
        naeherungswert = naeherungswert-(
            self.funktionswert(naeherungswert)/
            self.ableitungswert(naeherungswert)
        )
        naeherungswerte.append(naeherungswert)

    return naeherungswerte
