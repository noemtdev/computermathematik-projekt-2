Newton Verfahren

Das Newton-Verfahren ist eine Methode, um Näherungslösungen für Gleichungen zu finden. Es wird oft verwendet, um die Nullstellen einer Funktion zu finden, was bedeutet, die Stellen zu finden, an denen die Funktion den Wert null annimmt. Es ist nach Sir Isaac Newton benannt, einem berühmten Mathematiker und Physiker.

Hier ist eine Schritt-für-Schritt-Erklärung des Newton-Verfahrens:
1. Verstehen der Aufgabe: Angenommen, du hast eine Funktion f(x), und du möchtest wissen, an welcher Stelle x die Funktion den Wert null hat, also f(x) = 0.
2. Startpunkt wählen: Du beginnst mit einem Startpunkt  einer beliebigen Zahl. Idealerweise sollte dieser Startpunkt in der Nähe der Lösung liegen, aber das ist nicht immer notwendig.

5. Überprüfung der Genauigkeit: Du kannst überprüfen, ob deine Näherungslösung genau genug ist, indem du den Wert von f(x) an diesem Punkt betrachtest. Wenn f(x) nahe genug bei null liegt, hast du eine gute Näherung für die Lösung gefunden.

Ein Beispiel für die Umsetzung in Python (für ganzrationale Funktionen zweiten Grades) würde folgendermaßen aussehen:

class NewtonVerfahren:
    """
    Berechnung mit quadratischen Funktionen
    """
    def __init__(self, a:float, b:float, c:float):
        self.a = a
        self.b = b 
        self.c = c

    def funktionswert(self, x:float):
        a = self.a
        b = self.b
        c = self.c
        
        return a*x*x + b*x + c

    def ableitungswert(self, x:float):
        a = self.a
        b = self.b
        c = self.c

        return 2*a*x + b
    
    def verfahren(self, iterationen: int, naeherungswert: float) -> list:
        naeherungswerte = []
        for i in range(iterationen):
            naeherungswert = naeherungswert-(
                self.funktionswert(naeherungswert)/
                self.ableitungswert(naeherungswert)
            )
            naeherungswerte.append(naeherungswert)

        return naeherungswerte

newton = NewtonVerfahren(a=1, b=0, c=-2)
print(newton.verfahren(20, 3))

Um die Funktion zu verändern kann man mehr Variablen im Konstruktor (__init__) definieren, bzw die Funktion einfach überschreiben.
Hierbei sollte man nicht vergessen die Ableitung anzupassen.
Info: der letzte Wert in der Liste ist der genauste.
